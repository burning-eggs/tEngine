# **{ tEngine }**

tEngine is a ?rendering? engine written in [TS](https://www.typescriptlang.org/download) and [WebGL](https://get.webgl.org/)

## **{ Building tEngine }**
---

**DISCLAIMER: YOU NEED TO HAVE TYPESCRIPT INSTALLED**

You can build tEngine with the makefile:

```bash
$ make
```
